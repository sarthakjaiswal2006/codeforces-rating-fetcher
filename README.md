# 🚀 Codeforces Rating Checker

A simple and clean React application that allows users to check their Codeforces rating and rank in real-time.

---

## 📌 Features

* 🔍 Search any Codeforces user by username
* 📊 Displays current rating and rank
* 🎨 Color-coded ranks (Newbie → Legendary Grandmaster)
* ⚡ Fast and responsive UI
* ❌ Handles invalid usernames gracefully

--- 

## 🛠️ Tech Stack

* **Frontend:** React.js
* **Styling:** CSS
* **API:** Codeforces Public API

## ⚙️ Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/codeforces-rating-checker.git
```

2. Navigate to the project folder:

```bash
cd codeforces-rating-checker
```

3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm start
```
